Trang bị ngũ hành

Mỗi món có 3 dòng ẩn. Gộp 3 nguồn (ngũ hành nhân vật + 2 món ở cặp trước): bao nhiêu nguồn đúng hệ "mẹ" thì sáng bấy nhiêu dòng, theo thứ tự Ẩn 1 → Ẩn 2 → Ẩn 3.

Cách kích dòng ẩn

Món hệ E cần hệ "mẹ" (hệ sinh ra E). Nguồn: ngũ hành nhân vật + 2 món ở cặp đứng trước. 1 nguồn → Ẩn 1 · 2 nguồn → Ẩn 1+2 · đủ 3 → cả Ẩn 1, 2, 3.

Tương sinh: Mộc → Hỏa → Thổ → Kim → Thủy → Mộc  ·  Tương khắc (bỏ qua khi phối): Mộc khắc Thổ · Thổ khắc Thủy · Thủy khắc Hỏa · Hỏa khắc Kim · Kim khắc Mộc

Chiều ghép cặp có thể lệch chút theo phiên bản server — đối chiếu với bảng "Ngũ Hành" trong game của bạn.
